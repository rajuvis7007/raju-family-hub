Section 1

Getting Started

The Raju Family Hub is a private web app shared by four family members: Raju, Joshna, Amrita, and Shreya. Everything — photos, tasks, events, and notifications — is organized around these four people, each identified by a consistent color.

Member Color Where you'll see it
Raju Indigo Task dots, calendar chips, filter buttons
Joshna Rose / Pink Task dots, calendar chips, filter buttons
Amrita Emerald / Green Task dots, calendar chips, filter buttons
Shreya Amber / Orange Task dots, calendar chips, filter buttons

Navigation

The app has four main tabs accessible from the sidebar (desktop) or bottom navigation bar (mobile):

  • Dashboard — your family home screen
  • Photos — photo and video albums
  • Tasks — family to-do list
  • Calendar — monthly event calendar

The bell icon in the top-right corner (or sidebar bottom on desktop) controls browser notifications.

💡
Works on any device The app runs in any modern desktop or mobile browser. A native Android/iOS app is coming soon.

Section 2

Dashboard

The Dashboard is the first thing you see when you open the app. It gives you a quick snapshot of what's happening today and what's coming up for the family.

Widgets at a glance

  • Today's date — displayed prominently at the top of the page.
  • Family member cards — one card for each of the four members. Click any card to filter the widgets below to show only that person's tasks and events.
  • Tasks widget — shows all currently open tasks with their due dates. Each task row has a colored dot indicating who it's assigned to. Click any task to jump to the Tasks tab.
  • Coming Up widget — lists upcoming calendar events. Each event shows the date and a colored dot for the member who created it. Click any event to jump to the Calendar tab.

Filtering by member

  1. Click a family member card (e.g., "Amrita") on the Dashboard.
  2. Both the Tasks widget and the Coming Up widget instantly filter to show only Amrita's items.
  3. Click the same card again — or click "All" — to return to the full family view.
📋
Live data The Dashboard reflects the current state of your tasks and calendar in real time. Any task you complete or event you add will appear here immediately.

Section 3

Photos

The Photos tab is the family's shared photo and video library. Media is organized into albums — named collections that can represent events, trips, or any theme you choose.

Creating an album

  1. Go to the Photos tab.
  2. Click the New Album button (top right).
  3. Enter an album name (required).
  4. Optionally set an event date for the album.
  5. Choose who created it from the member list.
  6. Click Create. The empty album appears in the grid.
💡
Create first, upload later You can create an album now and add photos to it at any time using the "Add Photos" button inside the album.

Uploading photos & videos

The upload flow is a guided multi-step wizard. You can start it two ways: the Upload Photos button on the main Photos page, or the Add Photos button inside an open album.

Supported file types & limits:

  • Images: JPG, PNG, HEIC
  • Videos: MP4, MOV
  • Maximum file size: 200 MB per file

Upload wizard steps:

  1. Pick files — drag and drop files onto the upload zone, or click it to open your device's file picker. You can select multiple files at once.
  2. Choose or create an album — select an existing album from the list, or type a new name to create one on the spot.
  3. Upload — each file shows its own progress bar as it uploads. You can see how many files remain.
  4. Done — a confirmation screen appears. Click "View Album" to browse the newly uploaded photos.
⚠️
File size limit Files over 200 MB will be rejected before uploading. For very long videos, consider trimming them beforehand.

Browsing & viewing

  1. On the Photos main page, click any album card to open it. Photos and videos appear in a grid.
  2. Click any photo or video thumbnail to open the full-screen lightbox.
  3. Inside the lightbox:
    • Press → or ← arrow keys to navigate between items.
    • Press Escape to close and return to the album.
💡
Keyboard navigation Once the lightbox is open, you can swipe left/right on mobile or use the arrow keys on a desktop to flip through every photo in the album without closing the lightbox.

Deleting a photo

  1. Open an album and hover your cursor over the photo you want to remove.
  2. A × (delete) button appears in the corner of the photo.
  3. Click the × button. The photo is immediately removed from the album.
⚠️
Deletion is permanent Deleted photos cannot be recovered. Double-check before clicking the × button.

Section 4

Tasks

The Tasks tab is the family's shared to-do list. Tasks are assigned to individual members, can have due dates, and support file attachments.

Creating a task

  1. Go to the Tasks tab.
  2. Click the New Task button.
  3. Enter a title for the task (required).
  4. Pick a family member from the Assign to list.
  5. Optionally set a due date.
  6. Optionally attach files (see File attachments below).
  7. Click Save. The task appears in the Open section.

Managing tasks

Mark a task complete / incomplete:

  1. Click the circle checkbox to the left of any task.
  2. The task moves from the Open section to the Completed section (or back again).

Delete a task:

  1. Hover over the task row. A row of action icons appears on the right.
  2. Click the trash icon to permanently delete the task.
⚠️
No undo for deletion Deleted tasks and their attachments cannot be restored.

Filtering & due-date badges

Filter bar — at the top of the Tasks tab:

  • Click All to see every task.
  • Click a member's name to see only tasks assigned to that person.

Due-date badges appear on each task row:

BadgeMeaning
Overdue Due date has passed — needs immediate attention.
Today Due today — shown in bold amber.
Tomorrow Due tomorrow — shown in amber.
May 25 Future date — shown in gray.

File attachments

Tasks can carry supporting files — receipts, voice notes, photos, documents.

Supported types & limits:

  • Images & videos: up to 200 MB per file
  • Audio & PDF: up to 20 MB per file

Attach files when creating a task:

  1. Open the New Task dialog.
  2. Click the attachment area and pick one or more files.
  3. Save the task — files are uploaded along with it.

Attach files to an existing task:

  1. Hover over the task row to reveal the action icons.
  2. Click the paperclip icon.
  3. Pick files from your device. They are added to the task immediately.

Completed tasks

Completed tasks are shown below the Open section. To keep the page fast, only the 20 most recently completed tasks are loaded at first.

  • Scroll to the bottom of the Completed section.
  • Click "Load more completed tasks" to fetch the next batch of 20.
  • Repeat until all completed tasks are visible.

Section 5

Calendar

The Calendar tab shows a full monthly view of family events and task due dates. Day cells are color-coded so you can see at a glance what's happening and who's involved.

Navigating months

  • Click the ← (left arrow) to go to the previous month.
  • Click the → (right arrow) to go to the next month.
  • Click the Today button at any time to jump back to the current month.

Reading the calendar grid

Inside each day cell, you'll see two types of chips:

  • Filled color chip — a calendar event. The color matches the member who created it.
  • Light chip with a dot — a task due on that day. The dot color matches the task's assignee.

A member color legend is displayed below the calendar grid for quick reference.

Adding an event

You can add an event two ways:

  • Click the Add Event button in the top-right area of the Calendar tab.
  • Click a specific day cell to open the day panel, then click Add Event inside the panel (the date will be pre-filled).

Event form fields:

  1. Enter the event title (required).
  2. Confirm or change the date.
  3. Optionally set a start time and end time.
  4. Optionally add notes or a description.
  5. Pick the family member who is creating the event — this determines the chip color on the calendar.
  6. Click Save. The event chip appears on the correct day.

Day detail panel

  1. Click any day in the calendar grid.
  2. A detail panel opens on the right (or as a modal on mobile), listing:
    • Events — each with a delete button (trash icon) to remove it.
    • Tasks due — each with a checkbox to mark it complete or incomplete without leaving the Calendar.
  3. Click outside the panel (or press Escape) to close it.
💡
Quick task toggle You can mark tasks done directly from the day panel in the Calendar — no need to switch to the Tasks tab.

Section 6

Notifications

The app can send OS-level browser notifications to remind you about important items, even when the app is in a background tab.

What triggers a notification

  • Overdue tasks (past their due date)
  • Tasks due today
  • Events happening today
  • Events starting within 30 minutes
📋
No spam Each item fires a notification at most once per day. Reloading the page will not trigger duplicate notifications.

The bell icon — states

Bell appearanceMeaning
Indigo (filled) Notifications are enabled. You will receive alerts.
Outline (grey) Notifications not yet enabled. Click to enable.
Strikethrough Notifications are blocked by the browser.

A red badge on the bell shows the count of urgent items: overdue tasks + today's events.

Enabling notifications

  1. Click the bell icon in the top-right corner (or sidebar bottom on desktop).
  2. Your browser will ask for permission. Click Allow.
  3. The bell turns indigo — notifications are now active.

Re-enabling if blocked

If you previously clicked "Block" in the browser permission prompt, the bell shows a strikethrough. To re-enable:

  1. Open your browser's Settings.
  2. Go to Privacy & Security → Notifications (exact path varies by browser).
  3. Find the dashboard site in the list and change its permission to Allow.
  4. Reload the app — the bell should now be indigo.
⚠️
Browser permission required Notifications only work in modern desktop browsers that support the Web Notifications API. Safari on iOS may have limited support.
